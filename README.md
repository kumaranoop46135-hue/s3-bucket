# AWS S3 Bucket Setup Project

## Project Overview
This project explains how to create and configure an Amazon S3 bucket step by step using the AWS Management Console.  
Amazon S3 (Simple Storage Service) is an object storage service used to store files such as images, videos, backups, and static website files.

This project is part of my DevOps learning journey using AWS.

---

## What is Amazon S3?
Amazon S3 is a cloud storage service provided by AWS where data is stored in **Buckets** and files are stored as **Objects**.

Key features:
- Highly scalable storage
- Secure data storage
- Version control support
- Static website hosting
- Lifecycle management

---

## Tools & Services Used
- AWS Management Console
- Amazon S3
- IAM (Identity and Access Management)
- GitHub

---

## Steps to Create an S3 Bucket

### Step 1: Login to AWS Console
Login to your AWS account and search for **S3** in the services menu.

### Step 2: Create Bucket
- Click **Create Bucket**
- Enter a globally unique bucket name
